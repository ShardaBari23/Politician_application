# Voting System Application

This is a full-stack voting system application built with Node.js, Express, MySQL, and a client-side interface using HTML, CSS, and JavaScript.

## Project Structure

- **Client/**: Frontend files for Admin, Login, and User interfaces
- **Server/**: Backend server with API routes, database connections, and middleware

## Features

- User authentication and authorization
- Voter management
- Admin dashboard
- File uploads for voter data (Excel support)
- Secure password hashing with bcryptjs
- JWT token-based authentication

## Installation

1. Clone the repository
2. Navigate to the Server directory: `cd Server`
3. Install dependencies: `npm install`
4. Set up your MySQL database and configure environment variables in a `.env` file
5. Run the server: `npm start`

## API Endpoints

- `/api/auth`: Authentication routes (login, register, etc.)
- `/api/voters`: Voter management routes

## Technologies Used

- Backend: Node.js, Express.js, MySQL, JWT, bcryptjs, Multer
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
